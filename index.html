Redminator

Aggregate and edit multiple redmine instances or projects in one place.

Also my first take on node.js, backbone.js and jade.

Install

You need git and node.js installed. Then just issue these in you command line:

$ git clone git@github.com:jlogar/redminator.git
$ cd redminator
$ npm install

Use

To run Redminator just use:

$ node app.js
